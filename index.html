Latitude and the Weather

On this project we will be analysing how Latitude influences the weather in the World. Latitude ranges from -90 (South Pole) to 90 (North Pole) and 0 (Equator). You can read more about Latitude here Wikipedia

Summary

Utilizing Python (Pandas, Matplotlit and Numpy), OpenWeather API and CityPy API we were able to gather more than 500 random cities wolrdwide to see how latitude influeces the weather. The first step of this project, was to randomly selected numbers from -90 to 90 and place in a list for Latitudes then another list with random number from -180 to 180 for the Longitudes, these two list were then combined and we ended up with more than 500 pairs of Latitudes and Longitudes. With these pairs in hand, we queried the CityPy API for city names with correspondents Latitudes and Longitudes. And finaly we were able to retrieve weather date such as temperature, humidity, cloudiness and wind speed from the OpenWeather API and usigon Matplotlib plotted graphs to vizualize our findings.
